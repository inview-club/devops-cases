## Ход работы
### 1) Установка AppSec Portal
1. Клонируем репозиторий AppSec Portal

    ```git clone https://gitlab.com/whitespots-public/appsec-portal.git appsec-portal```

2. Настройка переменных окружения

    ```./set_vars.sh```


    ![alt text](<images/IMAGE 2025-12-21 21:00:27.jpg>)
    
3. Запускаем портал

    ```sh run.sh```

4. Создаем учетную запись суперпользователя
   
    ```docker compose exec back python3 manage.py createsuperuser --username admin```

5. Открываем портал в браузере вводим логин, пароль, ключ активации и видим что все работает

    ![alt text](<images/Снимок экрана 2025-12-19 в 17.24.26.png>)

### 2) Установка Auditor
1. Клонируем репозиторий 

    ``` git clone https://gitlab.com/whitespots-public/auditor.git auditor```

2. Запускаем Auditor

    ```docker compose up -d```  

3. Открываем в браузере, копируем Access Token и добавляем в файл .env

4. Перезапускаем

    ```docker compose down```

    ```docker compose up -d```

5. Все работает
